## Contributing

## Installing dependencies

```shell
make install
```
### Building SDKs

```
make build
```

### Generating SDK's

```
make generate
```

### Updating OpenAPI spec

All specifications are kept in `.openapi` folder:

https://github.com/redhat-developer/app-services-sdk-java/tree/main/.openapi
